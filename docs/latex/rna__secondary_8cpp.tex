{\footnotesize\ttfamily vector$<$ vector$<$ int $>$ $>$ optimal\+Matching (\begin{DoxyParamCaption}\item[{string}]{str }\end{DoxyParamCaption})}



Computes the optimal secondary structure of an RNA sequence using dynamic programming. 

This function implements the Nussinov algorithm to compute the optimal secondary structure of an RNA sequence using dynamic programming. The algorithm fills a 2D table {\ttfamily opt} where {\ttfamily opt\mbox{[}i\mbox{]}\mbox{[}j\mbox{]}} represents the maximum number of base pairs that can be formed by the substring from index {\ttfamily i} to index {\ttfamily j} in the RNA sequence.

The function first initializes the table for small substrings of length up to 4, where no base pairs can be formed. Then, it iterates over larger substrings and computes the maximum number of base pairs that can be formed by considering all possible pairs of indices {\ttfamily k} and {\ttfamily j} such that the nucleotides at those indices can form a base pair. It updates the {\ttfamily opt} table accordingly.

{\footnotesize\ttfamily void traceback (\begin{DoxyParamCaption}\item[{int}]{i,  }\item[{int}]{j,  }\item[{string}]{str,  }\item[{vector$<$ vector$<$ int $>$ $>$}]{dp,  }\item[{vector$<$ pair$<$ int, int $>$ $>$ \&}]{ans }\end{DoxyParamCaption})}



Performs traceback to reconstruct the optimal secondary structure from the dynamic programming table. 

This function performs a traceback on the dynamic programming table {\ttfamily dp} to reconstruct the optimal secondary structure of the RNA sequence. It takes the starting and ending indices {\ttfamily i} and {\ttfamily j}, the RNA sequence {\ttfamily str}, the dynamic programming table {\ttfamily dp}, and a reference to a vector {\ttfamily ans} that will store the bonded pairs of indices.

The function works recursively by checking if the maximum number of base pairs {\ttfamily dp\mbox{[}i\mbox{]}\mbox{[}j\mbox{]}} is the same as {\ttfamily dp\mbox{[}i\mbox{]}\mbox{[}j-\/1\mbox{]}}, in which case it recursively calls itself with {\ttfamily j-\/1} as the new ending index. Otherwise, it finds the index {\ttfamily k} such that the nucleotides at indices {\ttfamily k} and {\ttfamily j} can form a base pair, and the value {\ttfamily dp\mbox{[}i\mbox{]}\mbox{[}j\mbox{]}} is the sum of 1 and the maximum number of base pairs for the substrings from {\ttfamily i} to {\ttfamily k-\/1} and from {\ttfamily k+1} to {\ttfamily j-\/1}. It then adds the pair {\ttfamily (k, j)} to the {\ttfamily ans} vector and recursively calls itself with the updated indices.
